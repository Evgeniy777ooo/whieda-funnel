# Whieda Sales Funnel Bot

Telegram Mini App + сбор лидов в Google Sheets

## 🚀 Быстрый деплой на Render.com

1. Загрузи все файлы в GitHub репозиторий
2. На Render.com: New → Web Service → подключи репозиторий
3. Start Command: `python bot.py`
4. Environment Variables:
   - `BOT_TOKEN` — токен от @BotFather
   - `MINI_APP_URL` — URL GitHub Pages (например https://user.github.io/repo/)
   - `ADMIN_CHAT_ID` — ID чата для уведомлений
   - `GOOGLE_CREDENTIALS_JSON` — JSON-ключ сервисного аккаунта (в одну строку)

## 📱 Mini App
Загрузи `index.html` на GitHub Pages: Settings → Pages → Deploy from main
